# Hangman Game - Flask Web App

This is a Flask-based Hangman game where players guess letters to reveal a randomly chosen car brand name. The game provides a simple interface where users can guess letters and track their progress.

# 🚀 Features

Randomly selects a popular car brand name.

Allows players to guess letters one by one.

Tracks incorrect guesses (max 6 attempts before losing).

Displays progress with guessed letters.

Shows a win or lose message at the end.

# 🛠️ Technologies Used

Python

Flask

HTML/CSS (for templates)

#🎮 How to Play

Run the Flask server.

Click Start Game.

Enter letters to guess the word.

Win by guessing all letters or lose after 6 incorrect guesses.

# 🏗️ Installation & Setup

Clone the repository:

git clone https://github.com/yourusername/hangman-game.git
cd hangman-game

# Create a virtual environment (optional but recommended):

python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies:

pip install -r requirements.txt

Run the Flask app:

python app.py

Open your browser and go to http://127.0.0.1:8000/.
